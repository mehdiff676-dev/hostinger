Frequently Asked Questions

Find answers to common questions about our hosting platform

File Upload

What file types are supported?

We support PHP, Python (PY), HTML, CSS, JavaScript, JSON, TXT, and archive formats (ZIP, RAR, TAR.GZ). PHP and Python files are automatically executed on our servers.

What is the maximum file size?

You can upload individual files up to 300 MB. For larger files, please use our FTP service or split them into smaller archives.

How many files can I upload at once?

You can upload up to 50 files simultaneously. The total size of all files in a single batch should not exceed 1 GB.

Server Management

How do I restart a service?

Go to Server Management section, find the service you want to restart, and click the "Restart" button. Services will be temporarily unavailable during restart.

What PHP and Python versions are available?

We support PHP 7.4, 8.0, 8.1, and 8.2. Python versions 3.8, 3.9, and 3.10 are available. You can switch between versions in the server settings.

How do I check server resources?

The System Monitoring panel provides real-time CPU, RAM, storage, and network usage statistics. You can also set up alerts for resource thresholds.

Security

Is my data secure?

All data is encrypted with AES-256 encryption, stored on secure servers with daily backups, and protected by enterprise-grade firewalls.

How are file permissions handled?

PHP files have 644 permissions, Python files have 755. You can modify permissions through the File Manager or SSH access.
